## Setup Environment
conda activate --name main-ds python=3.12.0
conda activate main-ds
pip install pandas matplotlib seaborn jupyter streamlit

## Run Streamlit App
streamlit run dashboard.py
or you can also use
py -m streamlit run dashboard.py
